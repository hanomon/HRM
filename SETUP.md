# 🚀 빠른 시작 가이드

## 1단계: 의존성 설치

```bash
# 루트 디렉토리에서 실행
npm run install:all
```

이 명령어는 Backend와 Frontend의 모든 의존성을 자동으로 설치합니다.

## 2단계: 서버 실행

```bash
# Backend + Frontend 동시 실행
npm run dev
```

서버가 시작되면:
- 🌐 Frontend: http://localhost:5173
- 🔌 Backend API: http://localhost:3000

## 3단계: 샘플 데이터 추가 (선택사항)

테스트용 직원 데이터를 추가하려면:

```bash
cd backend
npm run seed
```

이 명령어는 3명의 샘플 직원을 데이터베이스에 추가합니다:
- 김철수 (개발팀 팀장)
- 이영희 (기획팀 대리)
- 박민수 (개발팀 사원)

## 4단계: 웹 애플리케이션 사용

### 💻 PC에서 테스트

1. 브라우저에서 http://localhost:5173 접속
2. "직원 관리" 메뉴에서 직원 등록
3. "기록 조회" 메뉴에서 기록 확인

### 📱 Android 태블릿에서 NFC 테스트

1. 태블릿과 PC가 같은 Wi-Fi 네트워크에 연결되어 있는지 확인
2. PC의 IP 주소 확인:
   - Windows: `ipconfig` 
   - Mac/Linux: `ifconfig` 또는 `ip addr`
3. 태블릿의 Chrome 브라우저에서 `http://[PC의 IP주소]:5173` 접속
   - 예: `http://192.168.0.100:5173`
4. "NFC 태깅" 메뉴로 이동
5. "NFC 스캔 시작" 버튼 클릭
6. NFC 카드를 태블릿에 태깅

## 🎯 주요 기능 사용법

### 직원 추가

1. 상단 메뉴에서 "직원 관리" 클릭
2. "직원 추가" 버튼 클릭
3. 정보 입력:
   - **NFC ID**: NFC 카드의 시리얼 넘버 (예: 04:A1:B2:C3:D4:E5:F6)
   - **이름**: 직원 이름 (필수)
   - 부서, 직책, 이메일, 전화번호 (선택)
4. "추가" 버튼 클릭

💡 **NFC ID 확인 방법**:
- Android 앱 "NFC Tools" 사용
- 또는 "NFC 태깅" 페이지에서 스캔 후 브라우저 콘솔에서 확인

### NFC 태깅으로 출퇴근 체크

1. "NFC 태깅" 메뉴로 이동
2. "NFC 스캔 시작" 버튼 클릭
3. NFC 권한 허용
4. 카드를 태블릿에 태깅
5. 자동으로 출근/퇴근 기록 생성

**자동 전환 로직**:
- 첫 태깅 → 출근
- 출근 후 다음 태깅 → 퇴근
- 퇴근 후 다음 태깅 → 출근
- 반복...

### 기록 조회 및 Excel 내보내기

1. "기록 조회" 메뉴로 이동
2. 날짜 범위 선택 (선택사항)
3. "검색" 버튼 클릭
4. "Excel 내보내기" 버튼으로 다운로드

## 🔧 문제 해결

### 포트가 이미 사용 중인 경우

Backend 또는 Frontend 포트를 변경하세요:

**Backend** (`backend/.env`):
```
PORT=3001
```

**Frontend** (`frontend/vite.config.ts`):
```typescript
server: {
  port: 5174,
  // ...
}
```

### NFC가 작동하지 않는 경우

1. ✅ Android 기기의 NFC가 켜져 있는지 확인
2. ✅ Chrome 브라우저 사용 확인
3. ✅ 같은 네트워크에 연결되어 있는지 확인
4. ✅ 방화벽이 5173 포트를 차단하지 않는지 확인

### 데이터베이스 초기화

```bash
cd backend
rm attendance.db
npm run dev  # 자동으로 새 데이터베이스 생성
npm run seed  # 샘플 데이터 추가
```

## 📱 Android 태블릿에서 PWA로 설치 (선택사항)
